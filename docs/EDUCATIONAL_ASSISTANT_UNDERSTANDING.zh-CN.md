# 教学助手升级：项目理解

## 目标

当前项目已经可以可视化 RAW 到 ISP 再到 RGB 的完整流程。下一步是让初学者不读文档也能开始使用。

这次升级会在应用内加入一个卡通讲解员，让它直接在工具界面中解释每个 ISP 阶段。用户打开页面后，可以选择测试案例，然后跟着讲解员一步步理解流程。

## 产品方向

应用应该像一个带友好导师的图像实验室：

- 讲解员解释当前阶段在做什么。
- 讲解员告诉用户应该观察图像和直方图的哪些变化。
- 讲解员给出一个简单的推荐操作。
- 用户可以用上一步 / 下一步按钮沿着 pipeline 学习。
- 点击任意阶段时，讲解员也会同步切换说明。
- 讲解员不替代工程参数，而是帮助用户理解这些参数。

## 初学者体验

初学者应该能回答：

- 这个阶段在做什么？
- 为什么相机 pipeline 需要这个阶段？
- 我应该先调哪个控件？
- 调完之后应该观察哪里？
- 我该如何进入下一步？

## 非目标

本次升级不会在应用内加入云端聊天机器人或大语言模型运行时。教学助手是确定性的、快速的、离线可用的，并且适合开源。

未来版本可以增加互动测验、界面语言切换和更深入的解释内容。
