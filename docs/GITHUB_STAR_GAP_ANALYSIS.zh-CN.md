# 距离高星 GitHub 项目的差距分析

本项目当前已经具备一个清晰的核心体验：用网页可视化展示 RAW Bayer 到 RGB 的 ISP 全流程，并通过交互式公式、测试案例、直方图和教学助手帮助初学者理解手机影像 pipeline。它已经适合作为个人作品集项目展示，但距离真正容易获得高 star 的开源项目，还需要继续在定位、可信度、工程质量、可扩展性和社区协作上补强。

## 当前优势

- **主题明确**：RAW -> ISP -> RGB 是图像算法工程师容易理解、也有稀缺性的方向。
- **第一屏可体验**：不需要安装复杂环境，打开网页就能看到 pipeline、案例图和调参结果。
- **学习路径完整**：包含 RAW、BLC、BPC、LSC、Demosaic、AWB、CCM、Denoise、Sharpen、Tone/Gamma 等阶段。
- **可视化程度较高**：每个阶段都有图像、直方图、统计值、公式说明和交互滑块。
- **开源交付完整**：已有 README、用户指南、技术设计、开发计划、测试脚本和 GitHub Pages。

## 与高星项目的主要差距

| 维度 | 当前状态 | 高星项目通常需要 |
| --- | --- | --- |
| 项目定位 | 教学型 ISP 可视化网页 | 更清晰的一句话价值主张、目标用户、真实使用场景 |
| 技术可信度 | 有基础 ISP pipeline 和 QA | 更完整的算法说明、边界条件、性能数据、误差或质量指标 |
| 可扩展性 | 单文件前端逻辑较多 | 模块化架构、插件式 pipeline、可替换算法实现 |
| 性能洞察 | 原来缺少阶段级性能数据 | 需要 latency、memory、bottleneck、优化建议等可观察性 |
| 用户案例 | 内置合成测试图 | 更多真实 RAW/DNG 样例、对比图、教程任务 |
| 开发者体验 | 可直接打开网页 | 更清晰的 npm scripts、CI、issue 模板、贡献指南和路线图 |
| 社区传播 | 标题有记忆点 | 更专业的截图/GIF、demo 视频、roadmap、英文传播文案 |

## 本次更新补强点

本次加入 **Performance Profiler 性能分析面板**，把项目从单纯“ISP 教学网站”进一步扩展为“影像 pipeline 可观察性工具”：

- 统计完整 RAW -> RGB pipeline 的总耗时。
- 统计每个 ISP 阶段的实际耗时和占比。
- 估算 RAW/RGB buffer、中间 Float32 buffer 和 canvas preview 的内存占用。
- 自动识别当前最大瓶颈阶段。
- 根据瓶颈、分辨率、降噪/锐化参数给出优化建议。

这使项目更接近 OPPO Color 派“AI 时代下的流畅性洞察和技术优化”方向：可以用真实图像处理 pipeline 作为计算负载，分析延迟、内存和优化方案。

## 下一阶段路线图

### 1. 算法可信度

- 增加更多 demosaic 算法：bilinear、edge-aware、Malvar-He-Cutler。
- 增加真实 RAW/DNG 解包或 metadata 解析能力。
- 增加 PSNR、SSIM、Delta E、灰卡误差等质量指标。
- 给每个 ISP stage 增加输入输出范围、假设和失败案例说明。

### 2. 性能和流畅性

- 将高耗时阶段迁移到 Web Worker。
- 支持 OffscreenCanvas 或分块处理。
- 增加只重算受影响下游阶段的 incremental pipeline。
- 增加不同分辨率下的 benchmark 表。
- 增加性能报告导出功能，生成 Markdown/JSON。

### 3. 教学体验

- 增加任务模式：例如“修复坏点”“校正暗角”“恢复白平衡”。
- 增加前后对比滑杆。
- 增加公式与真实图像联动的更多图示。
- 增加更真实的手机影像案例：夜景、逆光、人像肤色、室内混合光。

### 4. 开源工程化

- 增加 GitHub Actions 自动测试。
- 增加 issue 模板和 PR 模板。
- 增加架构图、性能截图和 demo GIF。
- 明确版本规划：0.1 教学稳定版、0.2 性能 profiler、0.3 真实 RAW 支持。

## 简历和报名包装建议

如果面向图像算法岗位，推荐标题：

> RAW-to-RGB ISP 可视化学习与调参平台

如果面向 Color 派“流畅性洞察和技术优化”，推荐标题：

> 面向移动影像 Pipeline 的流畅性洞察与优化分析工具

核心表达：

> 基于自研 RAW-to-RGB ISP pipeline，构建可交互的图像处理和性能洞察平台，支持阶段级图像预览、直方图统计、实时调参、耗时分析、内存估算和优化建议，用于理解移动影像处理链路中的画质变化与性能瓶颈。
