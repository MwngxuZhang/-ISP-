# PixelPipe ISP Lab

学习 ISP？PixelPipe 就够啦。

Learn camera ISP from RAW to RGB, one pixel at a time.

PixelPipe ISP Lab is a visual, browser-based playground for learning how sensor RAW data becomes a real RGB image. Pick a scene, drag a tuning control, and watch the RAW data, ISP stages, formulas, histograms, code snippets, and final image change together.

## Why It Is Fun To Learn With

- It runs in the browser. No heavy environment setup.
- Every ISP stage has a visual result, a formula explanation, and related source code.
- Built-in scenes cover color chart, low light, bad pixels, lens shading, portrait skin, night street, and backlit window.
- Real-time controls let beginners see what black level, white balance, demosaic, denoise, sharpen, tone mapping, and gamma actually do.
- A 3D learning assistant explains the pipeline in friendlier language.
- Bilingual UI and documentation make it easier to learn and present.

## Project Scope

PixelPipe ISP Lab is an educational RAW-to-RGB ISP lab, not a production camera firmware stack. Its goal is to make ISP concepts visible, testable, and easier to explain in interviews, portfolios, and beginner learning.
